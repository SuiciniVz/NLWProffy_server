# Getting Started

```bash
# Clone Repository
$ git clone https://github.com/SuiciniVz/NLWProffy_server.git

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
:eyes: Api running on http://localhost:3333/
